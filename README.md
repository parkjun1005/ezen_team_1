### 개발 기간
2024.07 ~ 2024.08

## 프로젝트 소개
- 리액트와 스프링부트, 데이터베이스를 연결하여 만든 프로젝트로서, localhost 주소를 이용해 백엔드와
프론트엔드 간의 데이터 연동을 구현하였습니다.
- 이 프로젝트는 요즘 흐름에 맞춰 증가하는 SNS와 인터넷 사용에 대응하여, 사용자 친화적인 예약 시스템과 정보
제공 플랫폼을 목표로 설계되었습니다
